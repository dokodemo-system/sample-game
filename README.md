# sample-game

ゴブレット風（3x3スタック型）オンライン対戦ゲームです。  
Node.js + Express + Socket.io で2人対戦のリアルタイム同期を行います。

## セットアップ

```bash
npm install
npm start
```

起動後に `http://localhost:3000` を開きます。

## 遊び方

1. ホストが「新しく部屋を作る」を押して5桁ルームIDを発行
2. ゲストが同じルームIDを入力して入室
3. 交互にコマを配置/移動して、最上段のコマで縦・横・斜めの3列を完成させた側が勝利

## ルール概要

- 盤面は3x3、各マスはスタック（重ね置き）構造
- 各プレイヤーの持ち駒は S/M/L が各2個
- 大きいコマのみ小さいコマの上に置ける
- 相手ターン中は操作ロック
- 切断後30秒以内に復帰しない場合は不戦敗
